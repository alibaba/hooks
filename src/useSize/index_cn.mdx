## API

```
const [ state, ref? ] = useSize(dom);
dom?: HTMLElement | (() => HTMLElement)
```

### 结果

| 参数     | 说明                                     | 类型       |
|----------|------------------------------------------|------------|
| state  | dom 节点的尺寸                             | { width: number, height: number }    |
| ref     | 当未传入任何参数时，将 ref 绑定给需监听的节点      | -        |

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| dom? | 可选项，如果未传入则会监听返回结果中的 ref，否则会监听传入的节点  | HTMLElement \| (() => HTMLElement) \| undefined | -      |
