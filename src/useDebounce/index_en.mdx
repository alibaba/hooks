# useDebounce

A hook that handle the debounce value.

## Examples

### Default usage

<JackBox jsCode={Demo1CodeJsx} tsCode={Demo1CodeTsx} demoName='Default usage' description='DebouncedValue will change after the input ends 500ms.'>
  <Demo1 />
</JackBox>

## API

```javascript
const debouncedValue = useDebounce(
  value: any,
  wait: number
);
```

### Params

| Property | Description                  | Type   | Default |
|----------|------------------------------|--------|---------|
| value    | value that requires debounce | any    | -       |
| wait     | wait time in milliseconds    | number | 1000    |
