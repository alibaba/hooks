获取元素的滚动状态。

## API
```
function useScroll(ref: RefObject<HTMLElement>): {
  left: number;
  top: number;
}
```

### Result

| 参数 | 说明     | 类型 |
|------|----------|------|
| left | 滚动容器的 scrollLeft 值 | number  |
| top | 滚动容器的 scrollTop 值 | number  |

### Params

| 参数    | 说明     | 类型                   | 默认值 |
|---------|----------|------------------------|--------|
| ref | 滚动容器的 ref | RefObject<HTMLElement> | -      |
