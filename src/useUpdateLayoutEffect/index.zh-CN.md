# useUpdateLayoutEffect

一个只在依赖更新时执行的 useLayoutEffect Hook。

## API

```javascript
useUpdateLayoutEffect(
  effect: () => (void | (() => void | undefined)),
  deps?: deps,
)
```

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| effect | 可执行函数  | () => (void | (() => void | undefined)) | -      |
| deps | 可选项，传入依赖变化的对象  | array \| undefined | -      |
