## API

```javascript
const ThrottledValue = useThrottle(
  value: any,
  wait: number
);
```

### Params

| 参数  | 说明                     | 类型   | 默认值 |
|-------|--------------------------|--------|--------|
| value | 需要节流变化的值         | any    | -      |
| wait  | 防抖等待时间，单位为毫秒 | number | 1000   |
