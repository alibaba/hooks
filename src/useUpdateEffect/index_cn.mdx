# useUpdateEffect

一个只在依赖更新时执行的 useEffect hook。

## API

```javascript
useUpdateEffect(
  effect: () => void,
  deps?: deps,
)
```

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| effect | 可执行函数  | function | -      |
| deps | 可选项，传入依赖变化的对象  | array \| undefined | -      |
