# @umijs/hooks

适用于中台场景的 react hooks 库

[![NPM version][image-1]][1] [![NPM downloads][image-2]][2]


## 安装

```
npm i @umijs/hooks --save
```

## 使用

```
import { useAsync } from '@umijs/hooks';
```

## 开发

```
$ git clone git@github.com:umijs/hooks.git
$ cd hooks
$ npm install
$ npm start
```
打开浏览器访问 http://127.0.0.1:8001/hooks
