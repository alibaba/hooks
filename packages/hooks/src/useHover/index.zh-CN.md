## API

```javascript
const isHovering = useHover(
  target, 
  {
   onEnter,
   onLeave
  }
);
```

### 结果

| 参数     | 说明                                     | 类型       |
|----------|------------------------------------------|------------|
| isHovering  | 判断鼠标元素是否处于 hover 元素                  | boolean    |

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| target | DOM 节点或者 Ref 对象  | (() => HTMLElement) \| HTMLElement \| React.RefObject | - |
| onEnter | 监听进入 hover  | ()=>void | -      |
| onLeave | 监听离开 hover  | ()=>void | -      |
