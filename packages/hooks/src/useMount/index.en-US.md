# useMount

A hook that executes a function after the component is mounted.

## API

```typescript
useMount(fn: () => void);
```

### Params

| Property | Description                 | Type         | Default |
| -------- | --------------------------- | ------------ | ------- |
| fn       | The function to be executed | `() => void` | -       |
