# useUnmountedRef

A Hook can be used to get whether the component is unmounted.

## API

```typescript
const unmountRef: { current: boolean } = useUnmountedRef();
```

### Result

| Property   | Description                        | Type                   |
|------------|------------------------------------|------------------------|
| unmountRef | Whether the component is unmounted | `{ current: boolean }` |
