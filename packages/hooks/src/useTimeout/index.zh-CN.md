## API

```typescript
useTimeout(
  fn: () => void,
  delay?: number | null
): fn: () => void;
```

### Params

| 参数  | 说明                                                                       | 类型                    |
| ----- | -------------------------------------------------------------------------- | ----------------------- |
| fn    | 待执行函数                                                                 | `() => void`            |
| delay | 定时时间（单位为毫秒）,支持动态变化，，当取值为 `undefined` 时会停止计时器 | `number` \| `undefined` |

### Result

| 参数         | 说明       | 类型         |
| ------------ | ---------- | ------------ |
| clearTimeout | 清除定时器 | `() => void` |
